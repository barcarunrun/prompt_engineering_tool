## 3. ページ構成（セクション一覧）

### セクション1: ヒーロー
- **見出し**: 「プロンプトの管理、まだスプレッドシートですか？」
- **サブコピー**: チームのプロンプトを一元管理。作成・テスト・バージョン管理をこれひとつで。
- **CTA**: 「無料で始める」ボタン
- **ビジュアル**: ツールのダッシュボードスクリーンショット or モックアップ

### セクション2: 課題提起
- **見出し**: 「LLM活用、こんな課題ありませんか？」
- 3〜4つの課題をアイコン付きで提示:
  1. プロンプトが属人化し、誰が何を変えたか分からない
  2. どのバージョンのプロンプトが最も効果的か判断できない
  3. LLMモデルごとの出力品質の比較に手間がかかる
  4. プロンプトのテスト作業が手動で非効率

### セクション3: ソリューション（機能紹介）
- **見出し**: 「すべて解決する、プロンプト管理プラットフォーム」
- 4つの主要機能をカード形式で紹介:

| 機能 | 説明 | アイコン案 |
|---|---|---|
| プロンプト作成・検証 | テンプレート変数対応。ワンクリックで実行＆結果確認 | ⚡ |
| バージョン管理 | 変更履歴を時系列管理。過去バージョンへのワンクリック復元 | 🕐 |
| 複数LLM対応 | OpenAI / Groq など複数プロバイダーを切り替えて比較 | 🔀 |
| 一括検証 | 複数テキストへの一括実行。結果をCSVダウンロード | 📊 |

### セクション4: チーム機能
- **見出し**: 「チームでの利用に最適化」
- ポイント:
  - Microsoft Entra ID によるSSO
  - ロールベースのアクセス制御（admin / member / viewer）
  - チーム単位でのプロンプト共有・管理

### セクション5: 料金
- **見出し**: 「完全無料で、すべての機能が使えます」
- 1カラムの料金カード:
  - **フリープラン: ¥0**
  - ✅ プロンプト管理・検証（無制限）
  - ✅ バージョン履歴
  - ✅ 複数LLM対応
  - ✅ 一括検証・CSVエクスポート
  - ✅ チーム管理・SSO
  - ※ LLM APIキーはお客様ご自身でご用意ください
- **補足**: 「なぜ無料？」→ 私たちはツール提供を通じてLLM活用の普及を支援し、より高度なDX推進をコンサルティングでお手伝いしています。

### セクション6: DXコンサルティング紹介
- **見出し**: 「LLMをもっとビジネスに活かしたい方へ」
- **サブコピー**: ツールだけでは解決できない、業務プロセスへのLLM組み込みを専門チームが支援します。
- サービス内容を3〜4つ:

| サービス | 内容 |
|---|---|
| プロンプト設計支援 | 業務に最適化されたプロンプトの設計・チューニング |
| LLM業務組込みコンサル | 既存業務フローへのLLM統合の企画・実装支援 |
| RAG / エージェント構築 | 社内ナレッジを活用したRAGシステムやAIエージェントの構築 |
| 社内研修・ワークショップ | エンジニア・非エンジニア向けのLLM活用トレーニング |

- **CTA**: 「まずは無料相談する」ボタン

### セクション7: 導入ステップ
- **見出し**: 「3ステップで始められます」
  1. **アカウント作成** — メールアドレスまたはMicrosoft Entra IDでサインアップ
  2. **チーム作成** — チームを作成し、メンバーを招待
  3. **プロンプト管理開始** — プロンプトを登録して検証スタート

### セクション8: FAQ
- Q: 本当に無料ですか？
  - A: はい。すべての機能を無料でご利用いただけます。LLM APIキーはお客様ご自身でご用意ください。
- Q: どのLLMに対応していますか？
  - A: OpenAI（GPT-5.4シリーズ）、Groq（Llama 3.xシリーズ）などに対応しています。
- Q: データのセキュリティは？
  - A: Azure上でホスティングされ、Microsoft Entra IDによる認証を採用しています。
- Q: コンサルティングだけの依頼も可能ですか？
  - A: はい、ツールをご利用いただかなくてもコンサルのみのご依頼も承ります。

### セクション9: CTA（クロージング）
- **見出し**: 「LLM活用の第一歩を、今日から」
- 2つのボタンを並列配置:
  - 「無料で始める」（プライマリ）
  - 「DX支援について相談する」（セカンダリ）

### セクション10: フッター
- 会社情報 / プライバシーポリシー / 利用規約 / お問い合わせ

## 5. 技術実装メモ

| 項目 | 方針 |
|---|---|
| ルート | `src/app/lp/page.tsx`（既存アプリ内に配置） |
| レイアウト | ダッシュボードレイアウトとは独立（ヘッダー・フッターのみ） |
| 認証 | 不要（公開ページ） |
| 問い合わせフォーム | 外部サービス連携（Google Forms / Calendly等）or 自前実装 |
| OGP / SEO | `metadata` でタイトル・説明・OG画像を設定 |
| アクセス解析 | Google Analytics / Microsoft Clarity 導入推奨 |
